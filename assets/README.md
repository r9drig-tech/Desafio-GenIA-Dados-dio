# Assets

Pasta pasta contém todos os recursos visuais que ilustram o funcionamento e a interface do AgentBot Analytics Neymar Jr. Career.

- Diagramas de Arquitetura: Representação visual do fluxo RAG (Retrieval-Augmented Generation), mostrando como o agente consulta os arquivos CSV e JSON antes de responder.
- Screenshots da Aplicação: Capturas de tela do chatbot em funcionamento no Streamlit, demonstrando consultas reais sobre gols e lesões.
- Mockups de Interface: Desenhos iniciais da experiência do usuário (UX) para o painel de analytics.
- Imagens do README: Logos, banners e ícones utilizados para organizar a documentação principal.

Destaque: Fluxo de Dados
O diagrama de fluxo abaixo (ou as imagens nesta pasta) explicam o diferencial do projeto:
Usuário ➔ Filtro de Pergunta ➔ Consulta Base de Dados (data/) ➔ Injeção de Contexto ➔ Resposta do LLM.
